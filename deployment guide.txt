Complete Deployment Guide: From Start to Live Updates
📋 Prerequisites
Node.js installed

GitHub account

Vercel account

🚀 PART 1: INITIAL SETUP & FIRST DEPLOYMENT
Step 1: Project Structure Setup
1.1 Verify your project structure:

text
frontend/
├── .env.development
├── .env.production
├── .gitignore
├── package.json
├── vite.config.js
├── index.html
└── src/
    ├── main.jsx
    ├── App.jsx
    ├── components/
    └── hooks/
1.2 Create environment files:

.env.development (for local development):

env
VITE_API_URL=http://localhost:3001/api
VITE_APP_TITLE=Pathfinder Academy - Development
VITE_DEBUG=true
.env.production (for live site):

env
VITE_API_URL=https://api.pathfinder.com/v1
VITE_APP_TITLE=Pathfinder Academy
VITE_DEBUG=false
1.3 Update package.json (use stable versions):

json
{
  "name": "frontend",
  "private": true,
  "version": "0.0.0",
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "lint": "eslint .",
    "preview": "vite preview"
  },
  "dependencies": {
    "react": "^18.3.1",
    "react-dom": "^18.3.1",
    "react-router-dom": "^6.28.0",
    "tailwindcss": "^3.4.17",
    "autoprefixer": "^10.4.20",
    "postcss": "^8.4.49"
  },
  "devDependencies": {
    "@vitejs/plugin-react": "^4.3.3",
    "vite": "^6.0.5"
  }
}
1.4 Create configuration files:

tailwind.config.js:

javascript
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
postcss.config.js:

javascript
export default {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
}
vite.config.js:

javascript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  server: {
    port: 3000,
    open: true
  }
})
1.5 Update src/index.css:

css
@tailwind base;
@tailwind components;
@tailwind utilities;
Step 2: Local Testing
2.1 Install dependencies:

bash
# Remove old node_modules if exists
rm -rf node_modules
rm -f package-lock.json

# Install fresh
npm install
2.2 Test locally:

bash
# Start development server
npm run dev
# Opens http://localhost:3000

# Test production build
npm run build
npm run preview
# Opens http://localhost:4173 (production preview)
Step 3: Git Setup
3.1 Initialize Git:

bash
# Initialize repository
git init

# Add all files
git add .

# Make initial commit
git commit -m "Initial commit: Pathfinder Academy website"

# Check status
git status
3.2 Create .gitignore:

text
# dependencies
node_modules/
npm-debug.log*

# production
dist/
build/

# environment variables
.env.local
.env.*.local

# misc
.DS_Store
*.tgz
Step 4: GitHub Repository
4.1 Create repository on GitHub:

Go to github.com

Click "+" → "New repository"

Name: pathfinder-frontend

Description: "Pathfinder Educational Institute Website"

Public repository

Don't initialize with README

4.2 Connect and push:

bash
# Add remote origin
git remote add origin https://github.com/YOUR_USERNAME/pathfinder-frontend.git

# Rename branch to main
git branch -M main

# Push to GitHub
git push -u origin main
Step 5: Vercel Deployment
5.1 Sign up for Vercel:

Go to vercel.com

Sign up with GitHub account

Complete onboarding

5.2 Deploy project:

Click "New Project"

Import Git Repository → Select pathfinder-frontend

Click "Import"

5.3 Configure project (Vercel auto-detects):

Framework: Vite

Build Command: npm run build

Output Directory: dist

Install Command: npm install

5.4 Add environment variables in Vercel:

Go to Project Settings → Environment Variables

Add production variables:

text
VITE_API_URL = https://api.pathfinder.com/v1
VITE_APP_TITLE = Pathfinder Academy
VITE_DEBUG = false
5.5 Deploy:

Click "Deploy"

Wait 2-5 minutes

Your site is live at: https://pathfinder-frontend.vercel.app

🛠️ PART 3: TROUBLESHOOTING
Common Issues & Solutions
Build fails on Vercel:

bash
# Test build locally first
npm run build

# If build fails, check:
# 1. Console errors
# 2. Missing dependencies
# 3. Syntax errors
Environment variables not working:

Check Vercel Environment Variables settings

Ensure variables start with VITE_

Redeploy after adding new variables

Styles not updating:

bash
# Restart dev server
npm run dev

# Or clear cache
rm -rf node_modules/.vite
npm run dev
Deployment stuck:

Check Vercel dashboard → Deployments

View build logs for errors

Cancel and retry if needed
